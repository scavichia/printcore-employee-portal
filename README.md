## 📖 Overview

PrintCore is an employee management platform designed to centralize internal workflows, reduce paperwork, and automate administrative processes.

The platform provides dedicated interfaces for employees, supervisors, and HR, allowing teams to manage routine tasks through a single web application. It also integrates approval workflows and notifications, replacing manual processes with digital solutions.

---

## Features

- Employee dashboard
- Supervisor dashboard
- Digital request forms
- Automated approval workflows
- Email notifications
- Employee attendance management
- Monthly Excel export
- Experience period reminders
- Promotion reminders
- Secure authentication
- Role-based access control
