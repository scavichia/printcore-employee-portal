# Database

## Overview

PrintCore uses Supabase as its backend service, providing authentication, PostgreSQL database, and file storage.

The database was designed to support multiple user roles, workflow automation, and document management while ensuring secure access control.

---

## Database Features

- User authentication
- Role-based access control
- Relational database
- File storage
- SQL queries
- Data integrity

---

## Main Entities

- Users
- Employees
- Supervisors
- Documents
- Notifications

---

## Security

- Authentication with Supabase Auth
- Role-based permissions
- Secure file storage
